**Aaron Levie** — CEO @box

在一个 agent 使用工具的频率比人类多 100 倍的世界里，软件走向无头化（headless）是必然的。而对很多软件来说，这实际上是这些平台潜在用例的巨大红利。

软件商业模型在传统上依赖于向公司特定职能的员工数收费，而且用户每天能做的事情本身就限制了软件的使用程度。这意味着你的技术相对于它实际能驱动的东西往往严重利用不足。

进入 agent 时代：agent 可以全天候工作、并行运行、跨系统串联任务。这是一件大事，因为现在 agent 能做到的事情远超人类——不再是一个个审阅合同，而是全部审阅；不再手动在营销系统和活动之间搬运数据，而是同时运行 10 倍数量的活动；不再因为人工步骤而在客户入职流程中受限，agent 来加速一切。

Agent 最终使用这些底层平台的频率远超人类曾经的使用量，这为平台开辟了之前无法触及的用例。

当然，不是每个软件市场在人类和 agent 之间都有同等的正和博弈，但可以说很大一部分系统级记录（如 Salesforce）可以被利用的程度远超今天。你的 Salesforce 数据可以被开发来执行远比现在多得多的客户定位和销售自动化。你的文档可以被转化为结构化数据，分析洞察，并自动化其他工作流。

现在你当然需要找到一种方式让这一切在商业上具有吸引力，但不难想象这些平台上 API 和 agent 消费产生的收入会成为丰富的收入来源——人类按席位付费，agent 按消耗量付费。这波趋势拥抱得好的公司会有很大的上涨空间。
